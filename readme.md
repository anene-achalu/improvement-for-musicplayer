### Features

Playback Controls: 

Full control over your music with Play/Pause, Stop, Next, and Previous buttons.

Volume Control:

Adjust the volume with a slider that provides numeric feedback (e.g., "70%") and a Mute/Unmute toggle.


Seek Functionality: 

Instantly jump to any part of a song by either clicking or dragging the progress bar.

Repeat Modes:

Cycle through three repeat options:

Repeat All (üîÅ):

The entire playlist will loop.

Repeat One (üîÇ):

The current song will repeat continuously.

No Repeat (faded üîÅ):

Playback stops after the last song.

Shuffle Mode (üîÄ):

Randomize the order of your playlist with a single click. The button highlights to indicate when shuffle is active.

Playlist Management:

Add Songs: Add songs individually or in bulk using the file chooser.

Drag & Drop: 

Drag music files directly from your file explorer onto the application window to add them to the playlist.

Remove Song: 

Remove the selected song from the playlist.

Save/Load Playlist: 

Save your current playlist to a file and reload it in a future session.

Real-Time Search: 

Instantly filter the playlist by song title or artist as you type in the search bar.

Favorite Songs:

Mark any song as a favorite by clicking the "‚ù§" icon next to it in the playlist.
Use the "Favorites" toggle button to view only your favorite tracks.

Lyrics Integration

Fetches lyrics automatically from the "lyrics.ovh" API.

Lyrics scroll smoothly in sync with the music for an interactive experience.

Keyboard Shortcuts: 

Control playback with your keyboard:
Spacebar: Play/Pause
Right Arrow: Next Song
Left Arrow: Previous Song
